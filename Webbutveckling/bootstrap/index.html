Om html och CSS.

HTML

HTML är det språk som man skriver i när man ska programera en hemsida, Den står för allt grundläggande som text och bilder, däremot så kan du inte ändra på saker som typsnitt, färg eller position på element med HTML, det behöver man skriva i CSS istället.

Element

Hela din kod bakom din hemsida är uppbyggd av element som är neslade på olika sätt. Ett element består oftast av två "taggar", en start tagg och en end tagg. Inehållet mellan de två taggarna är det som visas, ett exempel är att man skriver text mellan två p taggar för paragraph som bilden visar.

ID

ID låter dig sätta ett id på ett html element vilket kan låta dig påverka det i till exempel ett css dokument. I css skrivs ditt id som #dittID {}

Class

Class fungerar på ungefär samma sätt som ID, skillnaden är att du kan ha hur många element med samma klass som du vill. I css så skrivs klasser som .dinKlass {}
